youknownothing.js
=================

You know nothing, J.S.


## Install

```sh
$ npm install --save youknownothing.js
```
```sh
$ bower install --save youknownothing.js
```

## Usage

```js
var youknownothing = require('youknownothing.js');

youknownothing('Jon Snow');
//=> "You know nothing, Jon Snow"

```
